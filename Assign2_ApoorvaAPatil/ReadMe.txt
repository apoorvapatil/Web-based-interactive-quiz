Description:

Interactive quiz, interacts with user in selecting the level of the quiz. It shows the progress of the quiz along with the timer.
Also, you can rate the quiz application to help me improve on the application.

Questions are selected from easy and hard sets of questions randomly without any repetitions. For every question, the score is shown.
